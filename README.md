 This is a documentation tool where you can write snippets of code in the code cells and the output will be evaluated and displayed in the preview window next to them. In addition to that you can leave some comments in the text cells to the code you've written. The app supports installing npm packages, CSS styling and etc. 

###### Available Scripts

In the project directory, you can run:

#### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
